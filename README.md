<h1>Ejemplo de regresión lineal simple y múltiple en propiedades inmobiliarias de Argentina (2021)</h1>
<p><b>Fuente:</b> Properati</p>

<p>
  Este proyecto propone la aplicación de dos enfoques de Machine Learning
  (regresión lineal simple y regresión lineal múltiple) sobre un dataset con
  información inmobiliaria.
</p>

<p>
  Se utiliza Python como herramienta para construir modelos que aprendan e infieran
  reglas de negocio a partir de información etiquetada.
</p>

<p>
  A partir del análisis, se busca responder preguntas como:
</p>

<ul>
  <li>¿Qué tipo de propiedades son las más valoradas en el mercado?</li>
  <li>¿El valor de la propiedad aumenta al construir una habitación?</li>
  <li>¿Qué impacto tiene la superficie del inmueble sobre su precio?</li>
</ul>
